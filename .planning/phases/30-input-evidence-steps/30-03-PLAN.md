<objective>
Build the keystone Evidence step (Step 6) and the Media step (Step 7). Evidence is the source-gated core: repeatable source rows (URL + optional source-type select + optional title), the "minimum 2 independent public sources" rule surfaced up front, a live counter, the ≥2-LINKS gate (matching the server WEAK_SOURCE contract — files do NOT unlock advance), and the optional non-public lead note captured into `form.leadNote`. Media ports the legacy `/api/upload` flow for images/documents only (video hidden until Phase 33), with removable file cards, safety copy, and inline rejection of a personal social link.

Purpose: The 2-source gate is the platform's core value — "no source, no publication." Surfacing it up front and gating on links-only prevents the "passes client, server rejects" trap. The lead note replaces personal-anecdote-as-evidence without ever publishing or counting it.
Output: `src/components/wizard/EvidenceStep.tsx`, `src/components/wizard/MediaStep.tsx`.
</objective>
